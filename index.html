Frequently Asked Questions

Everything you need to know about EasyLock

Is my data safe?

expand_more

Yes! EasyLock uses AES-256 GCM encryption. Your passwords are stored securely using OS-native secure APIs.

What if I forget my password?

expand_more

Encrypted files cannot be recovered without the correct password. This ensures maximum security. We recommend using a password manager.

Which platforms are supported?

expand_more

EasyLock is compatible with Windows 10/11 and Linux (Ubuntu 20.04+). It integrates seamlessly with system file managers.

How fast is the encryption?

expand_more

EasyLock is optimized for speed. Most common files encrypt in under a second on modern systems.
